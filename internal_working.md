# How Python Source Code is Converted into Executable Code

The Python source code goes through several stages before producing the final output.

## Step 1: Writing the Source Code

The Python compiler/interpreter reads the Python source code written by the programmer in a code editor.

```python
print("Hello, World!")
```

This is the starting point of program execution.

---

## Step 2: Saving the Source Code

After writing the program, it is saved with a **.py** extension.

Example:

```text
hello.py
```

The `.py` file contains all the instructions written by the programmer.

---

## Step 3: Compilation to Bytecode

Python compiles the source code into **Bytecode**.

- The compiler checks for syntax errors.
- If no errors are found, Python generates a **.pyc** file.
- Bytecode is platform-independent and cannot be directly executed by the CPU.

Example:

```text
hello.py  →  hello.pyc
```

---

## Step 4: Execution by Python Virtual Machine (PVM)

The generated bytecode (`.pyc`) is sent to the **Python Virtual Machine (PVM)**.

The PVM:

- Reads the bytecode.
- Interprets and executes it instruction by instruction.
- Stops execution and displays an error message if an error occurs.

```text
Bytecode (.pyc)
       ↓
Python Virtual Machine (PVM)
```

---

## Step 5: Conversion to Machine Code

Inside the PVM, the bytecode is translated into **Machine Code**.

Machine code consists of binary digits:

```text
0 and 1
```

This code is understandable by the computer's CPU.

```text
Bytecode
    ↓
Machine Code (Binary)
```

---

## Step 6: CPU Execution

The CPU executes the machine code.

The instructions are processed and the desired output is displayed.

Example:

```python
print("Hello, World!")
```

Output:

```text
Hello, World!
```

---

# Flow Diagram

```text
Python Source Code (.py)
            ↓
        Compilation
            ↓
      Bytecode (.pyc)
            ↓
 Python Virtual Machine
            ↓
      Machine Code
            ↓
           CPU
            ↓
         Output
```

---

# Summary

1. Write Python code in a code editor.
2. Save the file with a `.py` extension.
3. Python compiler converts the source code into bytecode (`.pyc`).
4. The Python Virtual Machine (PVM) reads the bytecode.
5. The PVM converts bytecode into machine code.
6. The CPU executes the machine code and produces the final output.

> **Note:** Python is often called an interpreted language because the Python Virtual Machine (PVM) executes the bytecode instruction by instruction.
